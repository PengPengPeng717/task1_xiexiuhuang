## 一、原文硬性要求（无需确认，作对照）

| 项 | 要求 |
|----|------|
| 数据源 | 3 类：矿业新闻 / 关键矿产政策 / 价格 |
| 时间窗口 | 近 30 天 |
| 单类下限 | 每类 ≥ 200 条 |
| 总量下限 | 合计 ≥ 600 条 |
| 新闻 | 需全文 + 结构化抽取（非仅标题级） |
| 政策来源 | 中国稀土集团官网；澳洲 DISR 关键矿产战略（2023–2030） |
| 价格覆盖 | LME 铜/锌/镍；SHFE 锂；上海钢联铁矿石 |
| 问答 | 支持自然语言提问（示例：近 7 天澳洲锂出口政策有何变化） |
| 评测 | 20 条标准问答；检索前 5 条命中 + 答案忠于库内内容 |
| 交付说明 | 需书面记录：数据结构、字段、唯一标识、去重规则 |
